Connect Firebase Database

To sync data across all devices, paste your Firebase config below. This is free and takes 2 minutes to set up.

How to get your Firebase config:

1. Go to console.firebase.google.com

2. Click "Add project" → name it "business-tracker" → Create

3. Click "Web" icon (</>) → Register app

4. Copy the firebaseConfig values below

5. Go to Realtime Database → Create database → Start in test mode
